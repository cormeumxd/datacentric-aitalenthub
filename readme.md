AI-агент openclaw для автоматического поиска и сбора датасетов по любой теме

Агент использует Crawl4AI Skill для поиска и парсинга данных. Human-in-the-loop: агент предлагает варианты, вы выбираете - он пишет скрипт и забирает данные.

🎯 Задание 1: Поиск и сбор датасетов
Агент ищет датасеты по заданной теме, показывает результаты и после вашего подтверждения автоматически пишет скрипт для скрапинга и забирает данные.
