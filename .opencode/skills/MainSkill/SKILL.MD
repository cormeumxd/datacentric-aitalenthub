## About
Этот навык предназначен для управления основным циклом разработки ML-проекта по новостям: от сбора данных до обучения модели и развертывания агента. Он координирует работу других скиллов (Data Collection, Data Labeling, Data Quality, Active Learning) и обеспечивает интеграцию всех этапов в единый процесс. **Иди строго по шагам!!**
## What I do
1. Запуск Data Collection Skill для сбора данных по новостям
2. Запуск Data Quality Skill для анализа и чистки данных
3. Запуск Data Labeling Skill для разметки данных
4. Запуск Active Learning Skill для умного отбора данных и обучения модели
5. Координация всех этапов, обеспечение передачи данных между скиллами и контроль качества на каждом этапе
6. Финальный отчёт о проделанной работе и результатах
